## React Portfolio

## Description

A portfolio of work can showcase your skills and talents to employers looking to fill a part-time or full-time position. An effective portfolio highlights your strongest work as well as the thought processes behind it. This portfolio was built using React and Syntactically Awesome Style Sheets
